Обмеження вибірки

Для швидкої перевірки можна тимчасово обмежити кількість рядків. Для фінальних висновків краще повернути повний набір.

Описова статистика

Тут доречно переглянути базові числові характеристики перед побудовою графіків і моделей.

Розподіл колонки

Коли дивитися розподіл?

Розподіл допомагає побачити частоти, перекоси, дисбаланс класів і підозрілі значення перед кореляцією або регресією.

Парна кореляція

Як обирати колонки для кореляції?

Кореляція показує силу та напрям статистичного зв'язку між двома числовими ознаками. Pearson краще для приблизно лінійного зв'язку, Spearman - для монотонних залежностей і порядкових шкал.

Автоматичні висновки

Матриця кореляцій

Heatmap показує силу зв'язків між числовими колонками. Для читабельності система бере перші числові колонки поточного набору.

Лінійна регресія

Як обирати колонки для регресії?

Лінійна регресія оцінює, як зміна однієї числової ознаки пов'язана зі зміною числової цілі. Якщо зв'язок нелінійний або є багато викидів, результат потрібно трактувати обережно.

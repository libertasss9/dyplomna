Класифікаційні моделі

Оберіть категоріальну цільову колонку. Система порівняє Logistic Regression і Random Forest, а також сформує текстові висновки.

Як обрати цільову колонку для моделі?

Цільова колонка має містити обмежену кількість класів, а не унікальне значення для кожного рядка. Якщо класи закодовані числами, їх зміст варто описати у словнику колонок на сторінці профілю.

Ознаки для моделювання

Позначені колонки буде передано моделі. Джерело створеної цільової колонки, ідентифікатори та технічні поля система виключає автоматично.

Метрики

Як читати метрики моделей?
  • Accuracy показує загальну частку правильних прогнозів, але може бути оманливою при дисбалансі класів.
  • Balanced accuracy усереднює якість по класах і краще підходить, коли один клас значно частіший.
  • Macro F1 однаково враховує всі класи, тому корисна для дослідницького порівняння.
  • Weighted F1 враховує частоту класів і показує якість з урахуванням реального розподілу даних.
